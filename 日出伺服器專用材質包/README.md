# 日出伺服器專用材質包

合併自 **QualityArmory-21** 與 **IceTea** 飲品材質包。

## 內容

- QualityArmory：槍械、載具、配件等自訂模型
- IceTea：6 種冰茶飲品（藥水 CustomModelData 520000–520005）

## 使用方式

1. 將此資料夾壓縮成 ZIP（根目錄直接是 `pack.mcmeta` 和 `assets/`）。
2. 放進 Minecraft `resourcepacks` 並啟用。
3. 適配版本: Minecraft任意遊戲版本

## 遇到問題

1. 請檢視是否為伺服器設定問題
2. 請確認是否為其他插件(如假人插件)不適配材質包等問題
